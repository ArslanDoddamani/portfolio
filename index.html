Projects

  • Amazon_Clone

        The Amazon_Clone project recreates Amazon's interface using HTML and CSS exclusively. It features a responsive design, navigation bar with dropdown menus, product grid, detailed product pages, search functionality, and a shopping cart. The project aims to hone developers' skills in HTML and CSS fundamentals, responsive design, and e-commerce UI development. It serves as a practical exercise to understand web development principles while emulating a prominent online marketplace's user experience.

  • Netfilx

        The Netflix_Clone project replicates Netflix's interface using HTML and CSS alone. It showcases a responsive design, a navigation bar for easy browsing, a visually appealing grid layout for content display, and detailed pages for individual shows or movies. With search functionality and user-friendly controls, the project aims to enhance developers' proficiency in HTML and CSS, responsive design principles, and UI/UX development. It provides hands-on experience in crafting intuitive interfaces akin to a leading streaming platform, offering valuable insights into modern web design practices.

  • Tic_Tac_Toe

        This project is a simple implementation of the classic Tic Tac Toe game using HTML, CSS, and JavaScript. It features a basic grid layout for the game board, with players taking turns to mark spaces. JavaScript handles game logic, including checking for wins and enabling reset functionality. This project serves as a practical exercise to reinforce fundamental web development concepts such as DOM manipulation and event handling while creating an interactive game experience.
